Minimal setup requirements
==========================

1. Add in settings.php:
  $conf['mail_system']['default-system'] = 'PostageappDrupalMail';

2. Create template in your Postageapp account:
  TEMPLATE SLUG: drupal-default

and SAVE.

Use own module to add variables for Postageapp template - see postageapp.api.php on module folder.
